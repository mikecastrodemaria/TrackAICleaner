# Presets Guide

TrackWasher includes built-in presets optimized for specific AI music generators. Each preset adjusts all 12 processing stages to target the specific artifacts of that generator.

## Choosing a Preset

### Suno
Best for tracks generated by Suno AI. Suno uses a vocoder with strong HF artifacts and noticeable phase symmetry. This preset pushes HF smoothing and phase decorrelation higher.

```bash
python trackwasher.py input.wav output.wav --preset suno
```

### Udio
Optimized for Udio-generated tracks. Udio has particularly strong high-frequency comb patterns, so HF smoothing is set to 0.8. Other parameters are moderate.

```bash
python trackwasher.py input.wav output.wav --preset udio
```

### Generic AI
A balanced preset that works well for any AI generator (including lesser-known tools, covers AI, voice cloning, etc.). Good starting point if you're unsure which generator was used.

```bash
python trackwasher.py input.wav output.wav --preset generic
```

### Light Touch
Minimal processing for situations where you want subtle cleaning with minimal coloration. Useful for high-quality AI output that just needs a light wash.

```bash
python trackwasher.py input.wav output.wav --preset light
```

## Preset Values

| Parameter | Suno | Udio | Generic AI | Light Touch |
|---|---|---|---|---|
| Phase Decorrelation | 0.7 | 0.6 | 0.6 | 0.3 |
| Stereo Widening | 1.4 | 1.3 | 1.3 | 1.1 |
| HF Artifact Smoothing | 0.7 | 0.8 | 0.5 | 0.3 |
| Harmonic Enrichment | 0.3 | 0.2 | 0.25 | 0.1 |
| Micro-Timing Jitter | 0.4 | 0.3 | 0.3 | 0.2 |
| Spectral Noise Shaping | 0.3 | 0.25 | 0.2 | 0.1 |
| Multiband Compressor | 0.5 | 0.4 | 0.3 | 0.15 |
| Tape Saturation | 0.4 | 0.35 | 0.25 | 0.1 |
| Glue Compressor | 0.4 | 0.35 | 0.3 | 0.15 |
| Mid/Side EQ | 0.4 | 0.3 | 0.25 | 0.1 |
| Soft Clipper | 0.3 | 0.25 | 0.2 | 0.1 |
| Target LUFS | -14 | -14 | -14 | -14 |

## Overriding Preset Values

You can start from a preset and override individual parameters:

```bash
# Start from Suno preset but use louder mastering
python trackwasher.py input.wav output.wav --preset suno --lufs -11 --clip 0.4

# Start from Generic but disable tape saturation
python trackwasher.py input.wav output.wav --preset generic --tape 0
```

CLI overrides always take priority over preset values.
